# tforth

A minimal toy Forth interpreter written in C.

## What it does
Reads a Forth program from a file and evaluate it. Currently supports basic arithmetic operations on integers 

## Usage
``` bash
make 
make run

## Example 
5 5 *
output: 25

## Status
Work in progress. The goals is;
Complete arithmetic operations
Implement basic Forth words (dup, swap, drop)
Implement . (print top of stack)
